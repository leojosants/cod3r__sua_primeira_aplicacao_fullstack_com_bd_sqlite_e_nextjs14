# Aplicação Fullstack com Next e Sqlite

- [Next](https://nextjs.org/docs)
- [Prisma](https://www.prisma.io/)
- [Sqlite](https://www.sqlite.org/)
- [Tabler icons](https://tabler.io/icons)

## Instalar dependências

```next
  cd app-bd
  npm install
```

## Executar aplicação - port: 3000

```next
  cd app-bd
  npm run dev
```
